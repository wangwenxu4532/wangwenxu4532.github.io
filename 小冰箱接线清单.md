# 小冰箱接线清单

## 一、STM32 引脚 → 各模块（杜邦线）

| STM32 引脚 | 杜邦线 | 目标模块 | 目标引脚 |
|:----------:|:------:|----------|----------|
| **5V** | 公母线 | OLED | VCC |
| **5V** | 公母线 | DHT11 | VCC |
| **GND** | 公母线 | OLED | GND |
| **GND** | 公母线 | DHT11 | GND |
| **GND** | 公母线 | LR7843 模块 | GND（信号侧） |
| **GND** | 公母线 | 按键+ | G |
| **GND** | 公母线 | 按键- | G |
| **PB6** | 公母线 | OLED | SCL |
| **PB7** | 公母线 | OLED | SDA |
| **PA0** | 公母线 | DHT11 | DATA |
| **PA1** | 公母线 | 按键+ | S（信号） |
| **PA2** | 公母线 | 按键- | S（信号） |
| **PA6** | 公母线 | LR7843 模块 | SIG（信号输入） |

## 二、12V 大电流回路（螺丝刀拧，不经过面包板）

```
12V电源适配器
    │
    ▼ DC圆头插入
DC005母座
    │
    ├── 正极(+) ──拧螺丝── LR7843 电源输入正极
    ├── 负极(-) ──拧螺丝── LR7843 电源输入负极
    │
LR7843 模块
    │
    ├── 输出正极 ──拧螺丝── 制冷片红线
    ├── 输出负极 ──拧螺丝── 制冷片黑线 + 风扇黑线
    │
    └── 风扇红线 ──拧螺丝── 输出正极（和制冷片红线并一起）
```

---

## 三、供电总览

| 电源 | 给谁供电 | 怎么接 |
|------|----------|--------|
| USB（手机充电头） | STM32 开发板 | MicroUSB 线直插 |
| 12V 10A 适配器 | 制冷片 + 风扇 | 通过 DC005 → LR7843 |

## 四、引脚功能速查

| 引脚 | 功能 |
|:----:|------|
| PA0 | DHT11 温度传感器 |
| PA1 | 按键 +（温度加） |
| PA2 | 按键 -（温度减） |
| PA6 | PWM 输出 → MOS 模块控制制冷 |
| PB6 | OLED I2C 时钟线 SCL |
| PB7 | OLED I2C 数据线 SDA |
| 5V | OLED + DHT11 供电 |
| GND | 公共地（所有模块共地） |

---

## 五、接线顺序（推荐）

```
1. 面包板插好，供电轨接 5V 和 GND
2. OLED    ← 5V / GND / PB6 / PB7
3. DHT11   ← 5V / GND / PA0
4. 按键+   ← GND / PA1
5. 按键-   ← GND / PA2
6. LR7843 信号端 ← PA6 / GND（杜邦线）
7. LR7843 功率端 ← DC005 / 制冷片 / 风扇（螺丝刀拧）
8. STM32 USB 供电
9. 12V 电源插上（最后一步！）
```

---

## ⚠️ 注意事项

- **LR7843 的 GND 必须和 STM32 的 GND 用杜邦线连在一起（共地）**
- 12V 回路的所有接线用螺丝刀拧紧，不经过面包板
- 先插 USB 给 STM32 供电，确认 OLED 亮了，**最后再插 12V 电源**
- DHT11 和 OLED 供电用 **5V** 不是 3.3V（兼容性更好）
